## 📋 About

Hostel Management System (HMA) is a comprehensive platform designed to simplify and automate the operations of hostel management. It provides role-based access for admins, owners, and wardens with features like student management, room allocation, attendance tracking, fee collection, and automated notifications via WhatsApp and email.

### 💡 **Problem it solves**

Manual hostel management processes are time-consuming, error-prone, and lack real-time insights. HMA addresses this by centralizing all operations in one platform with role-based access, ensuring accountability, and automated reminders.

---

## ✨ Key Features

- 🎯 **Role-based Access Control**: Separate dashboards for Admins, Owners, and Wardens
- 👥 **Student & Room Management**: Easily manage student admissions and room allocations
- 📊 **Fee Management System**: Track fees, payments, and generate receipts
- 📅 **Attendance Tracking**: Monitor daily student attendance
- 💬 **Automated Reminders**: Send fee reminders via WhatsApp and Email
- 📈 **Analytics & Reports**: Generate financial reports and occupancy analytics
- 📄 **Document Generation**: Auto-generate admission contracts and receipts

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or later)
- PostgreSQL
- npm or yarn

### Installation

1. **Clone the repository**
```bash
git clone <your-repo-url
cd HMA-main
```

2. **Backend Setup**
```bash
cd backend
npm install
```

3. **Frontend Setup**
```bash
cd ../frontend
npm install
```

4. **Configure Environment Variables**
Create `.env` files in both `backend/` and `frontend/` directories (follow `.env.example` if available)

5. **Initialize Database**
```bash
cd ../database
# Import schema.sql into your PostgreSQL database
```

6. **Run the Application**

**Backend**:
```bash
cd ../backend
npm run dev
```

**Frontend**:
```bash
cd ../frontend
npm run dev
```

The app will be available at:
- Frontend: http://localhost:3000
- Backend: http://localhost:5000
